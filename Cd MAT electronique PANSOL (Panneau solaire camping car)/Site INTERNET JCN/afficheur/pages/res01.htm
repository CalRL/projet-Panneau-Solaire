L'afficheur MAT : matériels complémentaires

Réaliser les activités présentées nécessite de disposer d'un certains nombres de compléments, aussi la société MAT a répondu à la demande en commercialisant les éléments suivants. Pour les cartes électroniques, tous les plans nécessaires sont fournis ici afin que vous puissiez les réaliser vous même.

Si toutefois vous le souhaitez, les cartes sont disponibles à la vente sous 3 formes :

  • carte nue

  • carte nue avec sachet de composants

  • carte montée et testée

Les fichiers correspondants :

Un autocommutateur économique
Une carte de test pour liaison RS232
Une carte de test RS422, boucle de courant et entrées //
Un modem avec le même module que l'afficheur

L'autocommutateur peut-être utile si vous ne disposez pas de ce type d'équipement :

La carte de test RS232 permet de s'intercaler dans une liaison série afin de tester des configurations de câbles et de prélever ou d'injecter des signaux :

La carte de test RS422, boucle de courant et entrées // permet depuis les liaisons série et // d'un PC de générer les signaux RS422, 0 20 mA et //(0 24V) pour commander l'afficheur. Des bornes 4 mm permettent de visualiser les signaux.
